# RubyOnRails-Portfolio
Properties-Admin Application

# Technology
- Ruby 3.2.0
- Rails 7.0.7.2
- Javascript
- Bootstrap
- rspec-rails
- factory_bot_rails
- image_processing
- devise
- sassc-rails
- faker
- ransack
- will_paginate-bootstrap4
- service
- sqlite3 / mysql /postgree  
<br/><br/>
# Installation
bundle install
<br/>
rails db:migrate
<br/>
# Run Server
rails s
<br/>
# Run Test RSpec
bundle exec rspec
